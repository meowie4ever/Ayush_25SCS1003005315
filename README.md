# Hand Gesture Control System

A real-time computer vision project that lets users control the mouse using hand gestures captured through a webcam. The system uses **OpenCV**, **MediaPipe**, and **PyAutoGUI** to detect hand landmarks and map them to mouse actions.

## Features

- Move mouse with index finger
- Left click using fist gesture
- Right click using pinch gesture
- Scroll using two fingers
- Pause and resume control
- Reset gesture state

## Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- NumPy
